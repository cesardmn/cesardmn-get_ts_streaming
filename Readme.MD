# Get TS Streaming
Robot that downloads and renders streaming videos (.ts files) to .mp4

---

### Required
* ffmpeg


You must have ffmpeg installed.

The ffmpeg will render the video.

Get ffmpeg here: [ffmpeg.org](https://ffmpeg.org/download.html)

--- 

### Install
```bash
  $ git clone https://github.com/cesardmn/get_ts_streaming.git
  $ npm install
```
---

### Usage
```bash
  $ node index.js
  $ URL: 'paste here url video frame'
  $ Video name: 'set name for video'
```
